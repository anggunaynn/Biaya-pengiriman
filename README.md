# Biaya-pengiriman  Program ini digunakan untuk menghitung total biaya pengiriman paket berdasarkan:

Berat paket (kg)

Jarak tempuh pengiriman (km)

Ukuran paket (panjang, lebar, tinggi)

Dengan program ini, pengguna bisa mengetahui biaya total pengiriman secara otomatis tanpa perlu menghitung manual.
“Untuk menghitung dan menampilkan biaya pengiriman paket berdasarkan berat, jarak, dan volume barang dengan perhitungan otomatis.”
